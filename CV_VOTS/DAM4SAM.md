Q1: How to achieve multi-objct tracking with this model?

增强SAM2模型抵抗distractors的能力
**现有缺陷：** 传统的memory bank通过将包含分割目标的完整图像直接放入记忆库中来构建目标模型，这种方式不可避免地将当前帧的干扰物也隐式地编码进去了，容易引发视觉歧义 。

- **核心创新：** 干扰感知记忆模型 Distractor-Aware Memory (DAM)，将memory bank按照其在跟踪过程中的具体过程拆分成了两个独立的模块
	- **RAM - Recent Appearacne Memory 近期外观记忆:** 
		- **功能**：主要负责捕捉目标最新的外观变化，从而确保分割边界的准确性 。
		- **设计与更新**：它是一个先进先出（FIFO）队列，内部的数据带有时间戳（时间编码），以此来识别时间上更相关的特征 。RAM 每 5 帧更新一次 。为了防止记忆库被不包含目标的帧污染，当模型预测不到目标存在时（即预测掩码为空），RAM 将停止更新 。
		- Details:
			- 引入“目标感知”的安全更新机制，防止记忆污染：**
				-  **原始 SAM2 的缺陷**：原始 SAM2 模型会在视频的每一帧都进行记忆更新，甚至在目标完全被遮挡或离开画面时也会更新 。这会导致记忆库被没有包含目标的空白帧迅速填满，降低了目标外观模型的多样性，当目标重新出现时极易导致重检测失败和误差累积 。
		        - **RAM 的改进**：RAM 引入了判断机制，当模型预测不到目标（即预测的目标掩码为空）时，RAM 会直接停止更新 。这有效避免了错误背景信息对记忆库的污染。
		    - **降低更新频率，减少视觉冗余：**
			    -  **原始 SAM2 的缺陷**：由于相邻视频帧之间的图像信息高度相关，逐帧更新会造成记忆库中充斥着大量重复、冗余的视觉信息 。
		        - **RAM 的改进**：RAM 将更新频率从“逐帧更新”降低到了“每 5 帧更新一次” 。这既保证了能存入包含最新外观特征的帧，又更高效地利用了有限的记忆槽位，提高了注意力机制的定位效率 。
	         - **结构缩减与解耦，为抗干扰模块腾出空间：**
			    - **原始 SAM2 结构**：原始 SAM2 的记忆库固定保存 1 个初始帧和 6 个最近的历史帧，并对这 6 个历史帧全部应用时间编码 。
		        - **RAM 的结构**：在整个 DAM 架构中，RAM 作为一个先进先出（FIFO）队列，其容量被缩减为 3 个槽位 。它保留了原始模型中的时间编码功能（以便识别时间上距离当前帧更近的特征），从而专注负责提升目标分割的精度 。剩余的空间则被划分给了完全不使用时间编码的 DRM，用来专门存放干扰物信息以提升鲁棒性 。
	- **DRM - Distractor Resolving Memory 干扰物解析记忆：** 
		- **功能**：负责提供关键的锚点帧（Anchor frames），帮助模型在复杂场景下区分真正的目标和关键干扰物，以此大幅提升跟踪的鲁棒性和重检测能力 。
		- **设计与更新**：DRM 包含一个永久保留的初始监督帧，以及一个动态更新的锚点帧 FIFO 队列 。由于 DRM 的目的是为了解析干扰物并提供长期的先验参考信息，因此 DRM 内部的帧**不使用时间编码** 。
		- **利用分歧检测干扰**：研究发现，在跟踪器彻底被干扰物带偏之前，SAM2 的备选掩码往往已经捕捉到了这些干扰物 。基于此，作者设计了一种基于“假设发散度”的检测机制：通过对比主预测掩码和备选掩码之间的面积与空间差异，来自动判定当前帧是否存在潜在的干扰物 。
		- **稳定期安全更新**：为了避免把错误分割的目标当作干扰物存入记忆（这会导致严重的记忆损坏），模型只会在跟踪状态足够稳定（即 SAM2 输出的 IoU 置信度很高）且检测到明显的掩码分歧时，才会触发 DRM 的更新，将该帧作为“负面参考”存入记忆 。
