- **核心贡献 (Key Idea)：**
    - **双支路架构 (Dual-branch Propagation)**
		DeAOT 由两个平行的支路组成，它们共享相同的分层结构 ：
		- **视觉支路 (Visual Branch)**：负责匹配物体、收集过去的视觉信息并细化物体特征 。它学习更具辨别力的视觉嵌入，而不受特定物体 ID 的干扰 。
		- **ID 支路 (ID Branch)**：负责吸收从记忆帧传来的物体特定 ID 信息 。
		- **共享注意力**：ID 支路直接复用视觉支路计算出的注意力图（Attention Maps），因为物体的识别主要依赖于视觉特征而非 ID 索引 。
	- **门控传播模块 (Gated Propagation Module, GPM)**
		为了弥补双支路带来的额外计算开销，论文提出了更高效的 GPM 来替代 AOT 中的 LSTT 模块 ：
		- **单头注意力**：通过精心设计的门控机制，DeAOT 使用轻量级的**单头注意力**就能达到甚至超过多头注意力的效果，解决了 AOT 的效率瓶颈 。
		- **门控函数 (GP F unction)**：引入了门控嵌入和深度卷积（DW-Conv）来增强局部空间上下文的建模 。
		- **三种传播方式**：GPM 内部集成了**自传播 (Self-propagation)**、**长期传播 (Long-term)** 和**短期传播 (Short-term)**，全面覆盖了当前帧关联、历史记忆读取和相邻帧平滑处理 。
